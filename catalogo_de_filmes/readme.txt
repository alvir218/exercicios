exercicio: 
Implemente um sistema de avaliação de filmes.
O sistema deve permitir que usuários criem contas pessoais com login e senha. Cada usuário deve ser capaz de cadastrar novos filmes.
E filmes cadastrados podem receber notas de qualquer usuário.
Cada usuário só pode dar uma única nota por filme, mas pode alterar a nota caso deseje, ou remover sua nota. 
O sistema deve permitir a listagem de todos os filmes cadastrados e mostrar a média das notas dos filmes. 
O usuário que cadastrou um filme é o único autorizado a editar informações sobre aquele filme, ou remover o mesmo do sistema.

Todos os dados do programa devem ser salvos no disco para acessos futuros em formato de arquivos de texto. Em outras palavras ao finalizar o programa todos os dados serão salvos em um arquivo e ao iniciar todos os dados serão lidos desse mesmo arquivo(s).

O programa deve possuir uma classe no mínimo 3 classes: Usuario, Filme, Nota. Faça uso das práticas de orientação a objeto aprendidas e debatidas em sala.
Cada usuário deve ter ao menos os atributos: 
-string login
-string senha

Cada filme deve ter ao menos os atributos.
-string nome
-int anoDeLancamento
